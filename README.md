# 电子芯片类网站模板

这是一个响应式的电子芯片类企业网站模板，基于Bootstrap 3框架开发。

## 功能特点

- 响应式设计，支持PC端和移动端
- Bootstrap 3框架，兼容性好
- 轮播图展示
- 产品展示模块
- 新闻资讯模块
- 关于我们模块
- 底部留言表单
- 社交媒体链接
- 返回顶部功能

## 文件结构

```
├── index.html          # 主页面
├── css/
│   └── style.css      # 自定义样式
├── images/            # 图片文件夹
│   ├── logo.png       # 网站Logo
│   ├── banner1.jpg    # 轮播图1
│   ├── banner2.jpg    # 轮播图2
│   ├── icon1.png      # 优势图标1
│   ├── icon2.png      # 优势图标2
│   ├── icon3.png      # 优势图标3
│   ├── product1-6.png # 产品图片
│   ├── news1-3.jpg    # 新闻图片
│   ├── about-bg.jpg   # 关于我们背景图
│   ├── qrcode.png     # 微信二维码
│   └── f1-f5.png      # 社交媒体图标
└── README.md          # 说明文档
```

## 使用说明

1. 将所需的图片放入 `images` 文件夹
2. 在浏览器中打开 `index.html` 即可查看效果
3. 根据需要修改文字内容和链接

## 需要准备的图片

请准备以下尺寸的图片并放入 `images` 文件夹：

- logo.png - Logo图片（建议高度50px）
- banner1.jpg, banner2.jpg - 轮播图（建议1920x600px）
- icon1.png, icon2.png, icon3.png - 优势图标（建议尺寸75x75px）
- product1.png ~ product6.png - 产品图片（建议600x400px）
- news1.jpg ~ news3.jpg - 新闻配图（建议400x300px）
- about-bg.jpg - 关于我们背景图（建议1920x600px）
- qrcode.png - 微信二维码（建议200x200px）
- f1.png ~ f5.png - 社交媒体图标（建议20x20px）
